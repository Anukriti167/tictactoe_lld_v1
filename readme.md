🎮 Tic Tac Toe - Low Level Design (LLD)

📌 Overview

This project is a Tic Tac Toe game implementation in Java, designed with a strong focus on Low-Level Design principles and commonly used design patterns like:

* State Pattern → Manages game states (X turn, O turn, Win states)
* Strategy Pattern → Handles different player behaviors (e.g., Human player)

The goal of this project is to demonstrate how to structure a real-world system using clean architecture and separation of concerns, similar to what is expected in system design interviews.
